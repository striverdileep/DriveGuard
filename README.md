# DriveGuard
## SetUp
- Create a python virtual environment using the following command
```python
python3 -m venv venv
```
- Activate the Virtual environment with the below command
```
source ./venv/bin/activate
```
- Install the following dependencies
  - opencv-python
  - pytesseract
  - picamera2
  - spidev
  - Face_recognition
  - scipy
